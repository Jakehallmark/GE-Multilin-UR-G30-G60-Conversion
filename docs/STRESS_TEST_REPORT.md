## Methodology

Each iteration:

1. Staged a copy of the next source file (round-robin across URS or XML pools).
2. Ran a full G30 → G60 conversion against the bundled G60 base template.
3. Re-read the written output and verified every transferred register against expected values.

Verification tolerates decimal precision drift and benign coded-value display text differences. Any unexpected value change is flagged as a failure.
